# archinstall-contrib

Hosted one liner: `curl -sSL https://evoquus.com/s/co -o co && bash co archlinux archinstall` 

You can adapt this to anyone's [fork](https://github.com/archlinux/archinstall/forks?include=active&page=1&period=2y&sort_by=last_updated) and specify optional branch.

For example for my testing branch `curl -sSL https://evoquus.com/s/co -o co && bash co -cB h8d13 archinstall-patch testing`

`-c` for clean and `B` for no bytecode
